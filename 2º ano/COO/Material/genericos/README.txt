Exemplos relacionados ao recurso de Tipos Genéricos, introduzidos a partir do Java
1.5. Há dois subdiretórios com exemplos, cada um referente a um cenário distinto:

* pilha:

  Exemplo para motivar o aprendizado dos Tipos Genéricos. Envolvem a implementação 
  de uma estrutura de dados do tipo pilha. São apresentadas 3 versões de uma classe
  Pilha que evoluem de forma incremental rumo ao objetivo de maximizar a reutilização
  de código. O objetivo final é ter uma única implementação da estrutura, capaz de 
  manipular elementos de qualquer tipo, mas sem abrir mão da segurança da checagem 
  de tipos em tempo de compilação (algo que a versão intermediária - que trabalha 
  com referências para objetos - não consegue oferecer).

* metodosgenericos:
 
  Neste diretório são apresentados exemplos de métodos genéricos, dividos em dois
  arquivos fonte. No primeiro são apresentados exemplos de métodos genéricos que 
  manipulam arrays genéricos (arrays de referencias do tipo T). No segundo são 
  apresentados exemplos de métodos genéricos que manipulam Listas genéricas (tendo
  como base uma classe Lista fornecida junto com o exemplo).
